# 🌊 CodeStrix AI  
### AI-Powered Marine Hypoxia Early Warning System

CodeStrix AI is a satellite-driven marine intelligence platform that predicts coastal hypoxia (low oxygen levels) up to **5 days in advance** using AI-based environmental analysis.

Built to protect **small-scale fishers and fisheries authorities**, the system transforms reactive fish-kill response into proactive risk prevention.

---

## 🚨 Problem

Seasonal marine hypoxia events cause:
- Massive fish mortality
- ₹50–100 lakh losses per affected fishing zone
- Economic instability for small-scale fishers

Currently, no accessible 5-day predictive warning system exists for coastal communities.

---

## 💡 Solution

CodeStrix AI combines:

- 🌡 Sea Surface Temperature (SST)
- 🌿 Chlorophyll concentration
- 📊 Rolling environmental trends
- 🤖 Random Forest AI risk classifier

To classify coastal zones as:

- 🟢 **Safe**
- 🟡 **Warning**
- 🔴 **Severe**

With **~90% Severe Event Detection Recall**.

---

## 📍 Key Features

- GPS-based zone detection (50km radius)
- 5-day hypoxia prediction forecast
- Multi-language interface (English, Tamil, Hindi)
- AI-powered scientific explanation engine
- Risk dashboard for fisheries authorities
- Exportable reports (PDF ready)
